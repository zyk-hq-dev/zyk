INTERFACE

Claude via MCP

Create, trigger, and manage workflows directly in Claude. No separate tool to learn. And because Claude is the interface, you can go beyond a static task list — ask things like "show me all human-approval tasks that have been waiting longer than 24 hours."

VISIBILITY

Zyk UI

Workflow diagram, task list, and run history with direct links to the Hatchet dashboard for logs and tracing.

ENGINE

Hatchet

Durable execution under the hood. Retries, scheduling, and human-in-the-loop built in. Your workflows survive failures.
